# Mory

Mory is the AI assistant living in MoryFlow's right panel.

<!-- Screenshot placeholder: chat interface -->
![Chat Interface](/images/chat/chat-interface.png)

## What It Can Do

### Remembers What You've Said

Mory can read your notes, so it knows:

- What you've written
- What projects you're working on
- What ideas you have

**For example:**
```
You: Help me see what I did at work last week
Mory: Let me check your notes...
      [Reading] Work/Weekly/2024-01-15.md
      According to your records, you mainly completed...
```

## Managing Conversations

**Start new conversation**: Click **+** at the top of the chat panel

**Switch conversations**: Click the conversation title to see history

**Clear**: Can be cleared in settings

## Tips for Better Results

### Tell It Where to Look

```
Read "project-plan.md" and help me...
```

```
Search my notes for anything about "machine learning"
```

### Break Down Complex Tasks Step by Step

```
1. First look at my study notes
2. Summarize the key points
3. Help me create a review outline
4. Save it as a new note
```

### Be Clear About What You Want

```
List it in a table
```

```
Write about 500 words for the summary
```

## Things You Can Ask

**Look at current note**
```
Help me look at this note
```

**Edit notes**
```
Add a summary at the end of this note
```

**Create new notes**
```
Help me create a new note called "Meeting Notes"
```

**Organize files**
```
Help me categorize the notes in my "To Organize" folder by topic
```
