# Feature Overview

MoryFlow has three main parts: Notes Folder, Editor, and Mory.

## The Core Three

### Notes Folder

All your notes are stored in a regular folder, as Markdown files.

- Stored on your own computer
- Open with any editor you like
- Want to sync? Just put it in a cloud drive

[Learn more →](/en/features/vault)

### Editor

A smooth writing experience, kind of like Notion.

- What you see is what you get
- Type `/` to quickly insert things
- Supports code, tables, formulas
- Open multiple notes at once

[Learn more →](/en/features/editor)

### Mory

The AI assistant living inside MoryFlow.

- Reads your notes, understands you better over time
- Helps you write and edit
- Searches the web
- Breaks down complex tasks into steps and completes them

[Learn more →](/en/features/ai-assistant)

## Special Features

### Mory's Tools

Mory has 10+ built-in tools:

| What it does | Specifics |
|--------------|-----------|
| File operations | Read, write, edit, search, move, delete |
| Web access | Search info, fetch web pages |
| Run commands | Execute commands on your computer |
| Make plans | Break big tasks into small steps |
