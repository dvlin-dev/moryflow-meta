# Download & Install

MoryFlow supports macOS and Windows.

## System Requirements

**macOS**
- macOS 10.15 or later
- M-series chip or Intel both work
- 4GB RAM or more

**Windows**
- Windows 10 or later
- 64-bit system
- 4GB RAM or more

## Download

Get the latest version from [moryflow.com](https://moryflow.com).

<!-- Screenshot placeholder: download page -->
![Download Page](/images/settings/download-page.png)

## macOS Installation

1. Download the `.dmg` file
2. Double-click to open
3. Drag MoryFlow to the Applications folder
4. First time opening may show "developer cannot be verified"—go to System Settings → Privacy & Security → click "Open Anyway"

## Windows Installation

1. Download the `.exe` installer
2. Double-click to run
3. Follow the wizard and you're done

## First Launch

When you open MoryFlow, it'll ask you to choose a **folder** to store your notes.

:::tip Why choose a folder?
MoryFlow saves notes as regular text files (Markdown format) in the folder you pick. You can open them with any editor, and put the folder in a cloud drive to sync.
:::

<!-- Screenshot placeholder: folder selection -->
![Select Notes Folder](/images/vault/select-vault.png)

**Recommended locations:**

- **Mac**: `Documents/MoryFlow` or inside iCloud Drive
- **Windows**: `Documents\MoryFlow` or in OneDrive

Put it in a cloud drive folder for automatic sync to other devices. See [Multi-device Sync](/en/advanced/sync) for details.
