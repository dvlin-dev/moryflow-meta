# Settings

MoryFlow's settings panel lets you customize things the way you like.

## Opening Settings

Click the **gear icon** in the top right.

<!-- Screenshot placeholder: settings entry -->
![Settings Entry](/images/settings/settings-entry.png)

## What You Can Configure

### General

Basic configuration:

- **Notes folder path**: Where your notes are stored
- **Theme**: Light or dark
- **Language**: Interface language

### AI Models

Configure the AI models you want to use:

- Enter API keys for various providers
- Choose your default model
- Add custom models

[Learn about AI Model Configuration →](/en/settings/ai-models)

### Ollama Local Models

Want to use AI completely offline? Use Ollama:

- Configure Ollama connection
- Download and manage local models
- Fully offline, data never leaves your computer

[Learn about Ollama Configuration →](/en/settings/ollama)

### Custom Providers

Add other services compatible with the OpenAI protocol:

- Configure custom API endpoints
- Set up authentication
- Choose available models

[Learn about Custom Providers →](/en/settings/custom-provider)

### MCP Extensions

Add skills to Mory:

- Add and configure MCP servers
- View available tools
- Manage connection status

[Learn about MCP Extensions →](/en/features/mcp)

### Theme

Personalize the interface:

- Dark/light mode
- Follow system setting

[Learn about Theme Settings →](/en/settings/theme)

## Where Settings Are Stored

MoryFlow settings are stored locally and are not uploaded to the cloud.

Settings file location:
- **macOS**: `~/Library/Application Support/MoryFlow/`
- **Windows**: `%APPDATA%/MoryFlow/`

:::tip Tip
If settings get messed up, you can delete the settings files and restart the app—it will restore to defaults.
:::
