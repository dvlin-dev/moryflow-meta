# 常见问题

这里整理了大家常遇到的问题和解决办法。

## 安装与启动

### macOS 提示"无法验证开发者"

**问题**：第一次打开 MoryFlow 时，系统提示无法验证开发者身份。

**怎么解决**：
1. 打开「系统偏好设置」→「安全性与隐私」
2. 在「通用」标签里，点「仍要打开」
3. 或者右键点击应用，选「打开」

### Windows 安装后打不开

**问题**：装完了，点图标没反应。

**怎么解决**：
1. 看看是不是杀毒软件拦截了
2. 试试以管理员身份运行
3. 重新下载安装包，重装一次

### 启动时卡在加载界面

**问题**：打开后一直显示加载中。

**怎么解决**：
1. 等一等，第一次启动可能要 10-20 秒
2. 一直没反应就重启应用
3. 检查磁盘空间够不够
4. 删掉配置文件后重试（路径见下面）

**配置文件位置**：
- macOS：`~/Library/Application Support/MoryFlow/`
- Windows：`%APPDATA%/MoryFlow/`

## AI 配置

### API Key 无效

**问题**：配置了 API Key，但说无效。

**怎么解决**：
1. 确认完整复制了，没有多余空格
2. 检查 Key 有没有过期
3. 确认账户余额够
4. 确认选对了服务商

### AI 不回复

**问题**：发消息后，AI 没任何反应。

**怎么解决**：
1. 检查网络连接
2. 确认已经配置了 API Key
3. 确认选了可用的模型
4. 看看有没有错误提示

### 回复中断

**问题**：AI 回复到一半突然停了。

**可能原因**：
- 网络不稳
- 达到了模型的输出长度限制
- 服务商临时抽风

**怎么解决**：
1. 重新发一次
2. 把问题简化一下
3. 换个模型试试

### 需要翻墙吗？

**回答**：看你用哪家服务商。

| 服务商 | 是否需要 |
|--------|----------|
| OpenAI、Anthropic、Google | 需要 |
| DeepSeek、智谱、Moonshot | 不需要 |
| OpenRouter | 部分地区需要 |
| Ollama | 不需要（本地跑的） |

## 编辑器

### 中文输入法问题

**问题**：用中文输入法时有问题（比如重复字符）。

**怎么解决**：
1. 确保输入法是最新版
2. 换个输入法试试
3. 重启 MoryFlow

### 保存失败

**问题**：编辑后说保存失败。

**可能原因**：
- 文件被其他程序占着
- 磁盘空间不够
- 没有写入权限

**怎么解决**：
1. 关掉其他可能占用文件的程序
2. 检查磁盘空间
3. 确认对笔记文件夹有写入权限

### 格式显示不对

**问题**：Markdown 格式显示不正确。

**怎么解决**：
1. 检查 Markdown 语法对不对
2. 刷新一下页面
3. 重启应用

## 文件管理

### 笔记找不到了

**问题**：之前建的笔记找不到了。

**排查步骤**：
1. 用搜索功能找找
2. 看看是不是在其他文件夹里
3. 确认笔记文件夹路径对不对
4. 检查云盘有没有同步完
5. 看看系统回收站

### 文件冲突

**问题**：用云盘同步后出现冲突文件。

**怎么解决**：
1. 对比一下冲突文件的内容
2. 手动合并需要的内容
3. 删掉冲突副本
4. 看看 [多设备同步](/zh/advanced/sync) 怎么说的

### 删不掉文件

**问题**：删除文件时失败。

**怎么解决**：
1. 确认文件没有被其他程序占着
2. 检查有没有写入权限
3. 到文件管理器里手动删

## 性能问题

### 应用卡顿

**问题**：用起来感觉卡。

**怎么解决**：
1. 关掉不需要的其他应用
2. 减少同时打开的标签页
3. 文件太多的话，归档一下旧文件
4. 重启应用

### 内存占用高

**问题**：MoryFlow 占内存太多。

**怎么解决**：
1. 关掉不需要的标签页
2. 重启应用
3. 看看是不是有很多未关闭的对话

### Ollama 模型加载慢

**问题**：用 Ollama 时，第一次响应很慢。

**这是正常的**：因为要把模型加载到内存里。

**怎么优化**：
1. 用小一点的模型
2. 保持 Ollama 服务一直运行
3. 加内存

## 隐私与安全

### 我的数据安全吗？

**回答**：放心，我们很重视数据安全。

- **笔记数据**：完全存在本地，不会上传
- **AI 对话**：用云端模型时会发给 AI 服务商处理
- **配置信息**：存在本地
- **使用数据**：不收集任何使用数据

### API Key 会泄露吗？

**回答**：不会。

- API Key 存在本地配置文件里
- 不会上传到我们服务器
- 只在调用对应 AI 服务时使用

### 怎么完全离线用？

**方案**：用 Ollama 本地模型。

1. 装 Ollama
2. 下载本地模型
3. 在 MoryFlow 里选 Ollama 模型
4. 断网也能正常用
