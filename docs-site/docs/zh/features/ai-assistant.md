# Mory

Mory 就是住在 MoryFlow 右边的那个 AI 小助手。

<!-- 截图占位符：对话界面 -->
![聊天界面](/images/chat/chat-interface.png)

## 它能干嘛

### 记得住你说过的话

Mory 能读你的笔记，所以它知道：

- 你记了什么
- 你在忙什么项目
- 你有什么想法

**比如这样：**
```
你：帮我看看上周工作干了啥
Mory：让我看看你的笔记...
      [读取] 工作/周报/2024-01-15.md
      根据你的记录，上周主要完成了...
```

### 帮你做事情

不只是聊天，还能帮你把事搞定：

```
你：帮我准备下周的英语考试
Mory：好，让我看看你的学习笔记...
      [读取] 学习/英语/单词本.md
      [读取] 学习/英语/语法笔记.md
      [上网搜] 英语四级考试重点

      根据你的笔记和最新考点，我帮你制定了这个计划...
```

### 一切都是透明的

Mory 做事的时候，你能看到它在干嘛：

<!-- 截图占位符：工具调用展示 -->
![工具调用](/images/chat/tool-call.png)

- 读了哪个文件
- 搜了什么
- 改了什么

不会偷偷摸摸的。

## 对话管理

**开新对话**：点聊天面板顶部的 **+**

**切换对话**：点对话标题，可以看历史记录

**清空**：在设置里可以清掉

## 选模型

在聊天面板顶部可以换模型：

<!-- 截图占位符：模型选择 -->
![选模型](/images/chat/model-selector.png)

| 模型 | 特点 |
|------|------|
| Claude 3.5 Sonnet | 综合能力强 |
| GPT-4o | 响应快 |
| DeepSeek V3 | 便宜，中文好 |
| 本地模型 | 完全离线 |

详细配置看 [AI 模型配置](/zh/settings/ai-models)。

## 用得更顺的技巧

### 告诉它看哪里

```
读一下"项目计划.md"，然后帮我...
```

```
在我的笔记里搜搜关于"机器学习"的内容
```

### 复杂的事分步说

```
1. 先看看我的学习笔记
2. 总结一下重点
3. 帮我列个复习提纲
4. 存成新笔记
```

### 说清楚想要什么样

```
用表格列出来
```

```
写个 500 字左右的总结
```

## 可以让它做这些

**看当前笔记**
```
帮我看看现在这篇笔记
```

**改笔记**
```
在笔记最后加一段总结
```

**写新笔记**
```
帮我建个新笔记，叫"会议记录"
```

**整理文件**
```
帮我把"待整理"文件夹里的笔记按主题分一下类
```
