# Mory 的工具

Mory 不只是能聊天，它还有一堆工具可以用，能帮你做很多实际的事。

## 文件操作

### 读文件

```
帮我看看"工作/项目计划.md"里写了什么
```

### 写文件

```
帮我建个新笔记，叫"今天的想法"
```

会自动创建需要的文件夹，不用担心覆盖现有文件（它会先问你）。

### 改文件

```
在我这篇笔记开头加个目录
```

### 看文件夹里有什么

```
"工作"文件夹里都有哪些笔记？
```

### 搜文件

```
找找所有叫"会议"开头的笔记
```

```
在我的笔记里搜搜哪些提到了"机器学习"
```

### 移动文件

```
把"临时笔记.md"移到"归档"文件夹
```

### 删除

```
删掉"草稿.md"
```

删除的文件会进回收站，删错了还能捞回来。

## 上网

### 搜资料

```
搜一下 2024 年有什么值得关注的 AI 新闻
```

### 抓网页

```
帮我读一下这个网页：https://example.com/article
```

可以把网上看到的好文章内容抓下来存到笔记里。

## 跑命令

```
运行一下 npm install
```

:::tip 放心
Mory 跑命令之前会告诉你要执行什么，危险的操作会先问你。
:::

## 做计划

复杂的事情，Mory 会帮你拆成小步骤：

```
帮我制定一个两周的学习计划
```

它会把任务列出来，一步步执行，还会追踪进度。

## 它会组合着用

比如你说"帮我准备考试"，Mory 可能会：

1. 先读你的学习笔记
2. 上网搜最新的考点
3. 结合起来帮你写个复习计划
4. 存成新笔记

不用你一步步指挥，它自己会想办法。
