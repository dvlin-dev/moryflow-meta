# 编辑器

中间这块就是你写东西的地方，用起来有点像 Notion。

<!-- 截图占位符：编辑器界面 -->
![编辑器](/images/editor/editor-interface.png)

## 支持哪些格式

### 标题

```markdown
# 大标题
## 中标题
### 小标题
```

输入 `#` 加空格就变成标题了。

### 文字样式

| 效果 | 怎么打 | 快捷键 |
|------|--------|--------|
| **加粗** | `**文字**` | `⌘/Ctrl + B` |
| *斜体* | `*文字*` | `⌘/Ctrl + I` |
| ~~删除线~~ | `~~文字~~` | - |
| `代码` | `` `代码` `` | `⌘/Ctrl + E` |

### 列表

```markdown
- 这是无序列表
- 第二项

1. 有序列表
2. 第二项

- [ ] 待办事项
- [x] 已完成
```

### 引用

```markdown
> 引用的内容
```

### 代码块

````markdown
```javascript
console.log("Hello!")
```
````

会自动识别语言，加上颜色高亮。

### 表格

```markdown
| 名字 | 价格 |
|------|------|
| 苹果 | 5 元 |
| 香蕉 | 3 元 |
```

### 分隔线

```markdown
---
```

### 链接和图片

```markdown
[链接文字](https://example.com)
![图片说明](图片地址)
```

图片还可以直接拖进来。

## 斜杠命令

输入 `/` 会弹出一个菜单，可以快速插入各种东西：

<!-- 截图占位符：Slash 菜单 -->
![斜杠命令](/images/editor/slash-menu.png)

| 输入 | 插入 |
|------|------|
| `/h1` `/h2` `/h3` | 标题 |
| `/bullet` | 列表 |
| `/todo` | 待办清单 |
| `/code` | 代码块 |
| `/table` | 表格 |
| `/quote` | 引用 |
| `/image` | 图片 |

## 选中文字后

选中一段文字，上面会出现一个小工具栏：

<!-- 截图占位符：浮动工具栏 -->
![工具栏](/images/editor/floating-toolbar.png)

可以快速加粗、斜体、改颜色、加链接。

## 同时开多个笔记

可以像浏览器一样开好几个标签页：

<!-- 截图占位符：多标签页 -->
![标签页](/images/editor/tabs.png)

- 点文件就在新标签打开
- 点标签切换
- 点 × 关闭
- 拖动调整顺序

## 自动保存

写东西的时候不用按保存，MoryFlow 会自动帮你存。

右上角会显示状态：
- ✓ 已保存
- ○ 保存中
- ✗ 出错了

## 快捷键

**编辑**

| 功能 | Mac | Windows |
|------|-----|---------|
| 撤销 | `⌘ + Z` | `Ctrl + Z` |
| 重做 | `⌘ + Shift + Z` | `Ctrl + Shift + Z` |
| 全选 | `⌘ + A` | `Ctrl + A` |

**格式**

| 功能 | Mac | Windows |
|------|-----|---------|
| 加粗 | `⌘ + B` | `Ctrl + B` |
| 斜体 | `⌘ + I` | `Ctrl + I` |
| 代码 | `⌘ + E` | `Ctrl + E` |

**导航**

| 功能 | Mac | Windows |
|------|-----|---------|
| 快速找文件 | `⌘ + P` | `Ctrl + P` |
