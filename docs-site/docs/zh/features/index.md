# 功能概览

MoryFlow 主要有三块：笔记文件夹、编辑器、还有 Mory。

## 三大核心

### 笔记文件夹

你的笔记都存在一个普通文件夹里，都是 Markdown 文件。

- 存在你自己电脑上
- 用什么编辑器都能打开
- 想同步就放云盘里

[详细了解 →](/zh/features/vault)

### 编辑器

写起来很顺手的编辑器，有点像 Notion。

- 所见即所得
- 输入 `/` 快速插入
- 代码、表格、公式都支持
- 可以同时开好几个笔记

[详细了解 →](/zh/features/editor)

### Mory

住在 MoryFlow 里的 AI 小助手。

- 能读你的笔记，越聊越懂你
- 能帮你写东西、改东西
- 能上网查资料
- 能把复杂的事拆成小步骤做完

[详细了解 →](/zh/features/ai-assistant)

## 特色能力

### Mory 会用的工具

Mory 有 10 多种内置工具：

| 能干嘛 | 具体的 |
|--------|--------|
| 文件操作 | 读、写、改、搜、移动、删除 |
| 上网 | 搜资料、抓网页 |
| 跑命令 | 在你电脑上执行命令 |
| 做计划 | 把大任务拆成小步骤 |

[详细了解 →](/zh/features/ai-tools)
