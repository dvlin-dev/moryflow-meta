# 笔记文件夹

你的笔记都存在这个文件夹里。

## 就是个普通文件夹

MoryFlow 的笔记不是什么特殊格式，就是普通的 Markdown 文件，存在你指定的文件夹里。

这意味着：

- 用 VS Code、Typora 或者记事本都能打开
- 哪天不想用 MoryFlow 了，笔记还是你的
- 想迁移到别的软件？直接复制过去就行
- 想用 Git 管理版本？完全可以

## 左边的文件列表

打开 MoryFlow，左边就是你的文件夹结构。

<!-- 截图占位符：文件树 -->
![文件列表](/images/vault/file-tree.png)

**显示规则**
- 只显示 `.md` 文件和文件夹
- `.` 开头的隐藏文件不显示
- 文件夹在前，按名字排序

**展开收起**
- 点文件夹前面的箭头
- 或者双击文件夹

## 常用操作

### 新建笔记

- 点左上角的 **+** 按钮
- 或者右键 → 新建笔记
- 输个名字就行，不用打 `.md`

### 新建文件夹

右键 → 新建文件夹 → 输名字

### 改名字

- 右键 → 重命名
- 或者选中后按 `F2`

### 移动

直接拖过去，或者右键 → 移动到...

### 删除

右键 → 删除

:::tip 放心删
删除的文件会进系统回收站，删错了还能找回来。
:::

### 在 Finder/资源管理器里打开

右键 → 在 Finder 中显示（Mac）或 在资源管理器中显示（Windows）

## 搜索

### 快速找文件

按 `⌘ + P`（Mac）或 `Ctrl + P`（Windows），输文件名的一部分就能找到。

### 搜内容

让 Mory 帮你搜：

```
帮我找找哪些笔记提到了"项目计划"
```

## 换个文件夹

如果你有好几个文件夹想切换：

1. 打开设置
2. 在「常规」里找到笔记文件夹路径
3. 点「更换」选一个新的

## 怎么组织比较好

### 推荐的结构

```
我的笔记/
├── 工作/
│   ├── 项目A/
│   └── 项目B/
├── 学习/
│   ├── 英语/
│   └── 编程/
├── 生活/
│   ├── 旅行/
│   └── 读书/
└── 日记/
```

### 取名建议

- 名字要能看出是啥
- 想按时间排？加个日期前缀：`2024-01-15 会议记录`
- 别用这些字符：`/ \ : * ? " < > |`

更多整理技巧看 [文件组织](/zh/advanced/file-organization)。
