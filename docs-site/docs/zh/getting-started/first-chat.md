# 和 Mory 聊一聊

现在来试试 MoryFlow 的核心功能——Mory，你的 AI 小助手。

## 先配置一下 AI

聊天之前，要先告诉 MoryFlow 用哪个 AI 服务。

### 推荐：OpenRouter

[OpenRouter](https://openrouter.ai) 是个 AI 服务聚合平台，一个账号就能用各种模型，挺方便的。

1. 去 [openrouter.ai](https://openrouter.ai) 注册
2. 在 Dashboard 创建一个 API Key
3. 打开 MoryFlow 设置（右上角齿轮图标）
4. 找到「模型管理」
5. 把 API Key 填到 OpenRouter 那一栏

<!-- 截图占位符：配置 API Key -->
![配置 API Key](/images/settings/api-key-config.png)

### 选个模型

配好之后，在右边聊天面板顶部选一个模型：

<!-- 截图占位符：选择模型 -->
![选模型](/images/chat/model-selector.png)

推荐：
- **Claude 3.5 Sonnet**：综合能力强，日常够用
- **GPT-4o**：响应快
- **DeepSeek V3**：便宜，中文好

## 开始聊

在右边的输入框打字，回车发送。

<!-- 截图占位符：对话界面 -->
![聊天界面](/images/chat/chat-interface.png)

### 试试这些

**随便问点什么：**
```
Markdown 是什么？
```

**让 Mory 看看你的笔记：**
```
帮我看看刚才写的笔记，总结一下
```

**让它帮你写东西：**
```
帮我写一篇关于时间管理的笔记
```

**让它上网查资料：**
```
搜一下最近有什么好看的电影
```

## Mory 能做什么

它不只是聊天，还能：

**读写你的笔记**
- 看笔记内容
- 帮你写新笔记
- 在所有笔记里搜东西
- 帮你整理、移动文件

**上网找资料**
- 搜索最新信息
- 抓取网页内容

**执行任务**
- 跑一些命令
- 把复杂的事拆成小步骤做完

## 看看它在做什么

Mory 工作的时候，你能看到它的"思考过程"——读了哪个文件、搜了什么、做了什么改动。

<!-- 截图占位符：工具调用展示 -->

一切都是透明的，你知道它做了什么。

## 管理对话

- **新对话**：点聊天面板顶部的 **+**
- **切换对话**：点对话标题，可以看历史
- **清空**：在设置里可以清掉当前对话
