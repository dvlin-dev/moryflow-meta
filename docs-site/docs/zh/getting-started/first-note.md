# 写第一篇笔记

选好文件夹后，你就进入主界面了。来写点东西试试。

## 新建笔记

有几种方式：

**点按钮**
1. 左边文件夹列表顶上有个 **+** 按钮
2. 点它，选「新建笔记」
3. 输个名字，回车

<!-- 截图占位符：新建笔记按钮 -->
![新建笔记](/images/vault/new-note-button.png)

**右键菜单**

在左边空白处右键 → 新建笔记 → 输名字

**快捷键**

- Mac：`⌘ + N`
- Windows：`Ctrl + N`

## 开始写

笔记建好后，中间的编辑区就可以直接写了。

### 斜杠命令

输入 `/` 会弹出快捷菜单，可以快速插入各种东西：

<!-- 截图占位符：Slash 命令菜单 -->
![斜杠命令](/images/editor/slash-menu.png)

| 输入 | 插入 |
|------|------|
| `/h1` `/h2` `/h3` | 标题 |
| `/bullet` | 列表 |
| `/todo` | 待办清单 |
| `/code` | 代码块 |
| `/table` | 表格 |

### Markdown 语法

如果你熟悉 Markdown，直接打也行：

```markdown
# 大标题
## 小一点的标题
- 列表项
- [ ] 待办事项
> 引用
**加粗** *斜体*
```

### 选中文字后

选中一段文字，会出现一个小工具栏，可以快速加粗、斜体、改颜色这些。

<!-- 截图占位符：浮动工具栏 -->
![工具栏](/images/editor/floating-toolbar.png)

## 自动保存

放心写，MoryFlow 会**自动保存**，你不用管。

右上角会显示保存状态：
- ✓ 已保存
- ○ 保存中
- ✗ 出错了

## 整理笔记

### 建文件夹

右键左边空白处 → 新建文件夹

### 移动笔记

直接拖过去就行，或者右键 → 移动到...

### 改名字

右键 → 重命名，或者选中后按 `F2`
