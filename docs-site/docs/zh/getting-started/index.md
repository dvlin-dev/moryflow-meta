# MoryFlow 是什么

MoryFlow 是一款笔记应用，里面住着一个叫 **Mory** 的 AI 助手。

它不是那种问一句答一句的聊天机器人，而是能读你的笔记、记住你说过的话、帮你把事情搞定的**小伙伴**。

## 和普通 AI 有什么不同

### 它记得住

普通的 AI 助手每次聊天都是"失忆"的——你得反复告诉它你是谁、你想要什么。

Mory 不一样。它能读你的笔记，知道你在忙什么、喜欢什么。**聊得越多，越懂你**。

### 笔记存在你电脑上

你写的东西都是普通的文件，就存在你自己电脑的文件夹里：

- 不上传到任何服务器
- 断网也能正常用
- 想同步到云盘？把文件夹放进去就行
- 哪天不想用了，笔记还是你的

### 能帮你把事搞定

Mory 不只是聊天，它能：

- 读你的笔记，帮你整理、总结
- 上网搜资料，找到有用的信息
- 帮你写东西、做计划
- 把复杂的事情拆成小步骤，一步步做完

## 界面长什么样

打开 MoryFlow，你会看到三栏：

| 位置 | 干嘛用的 |
|------|----------|
| **左边** | 你的笔记文件夹，所有笔记都在这 |
| **中间** | 编辑器，写笔记的地方 |
| **右边** | Mory，随时可以找它聊 |

## 适合什么人

- **学生**：整理笔记、准备考试、做学习计划
- **上班族**：写周报月报、策划活动、管项目
- **写作的人**：攒素材、理思路、找灵感
- **在意隐私的人**：笔记就在自己电脑上，踏实
