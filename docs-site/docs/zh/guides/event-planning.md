# 用 MoryFlow 策划活动

让 Mory 当你的策划搭档，从灵感到方案，高效搞定。

## 适用场景

要策划个活动（公司年会、团建、生日派对啥的），想要：

- 借鉴好案例，找找灵感
- 按预算做可行方案
- 考虑周全，别漏东西
- 快速出一份完整文档

## 准备工作

### 1. 建个活动文件夹

```
工作/
├── 活动策划/
│   ├── 2023年会总结.md
│   ├── 团建活动记录.md
│   └── 活动模板.md
└── 预算/
```

### 2. 把以前的活动记下来

包括：

- 活动流程
- 实际花了多少钱
- 效果怎么样
- 踩过什么坑

**2023年会总结.md 示例：**
```markdown
# 2023年度年会总结

## 基本信息
- 时间：2023年12月25日
- 地点：XX酒店
- 人数：80人
- 预算：5万元

## 活动流程
1. 18:00 签到入场
2. 18:30 领导致辞
3. 19:00 晚宴开始
4. 20:00 节目表演
5. 21:30 抽奖环节
6. 22:00 结束

## 效果反馈
- 好评：节目精彩，抽奖环节气氛好
- 改进：时间有点赶，可以多加点互动

## 花费明细
- 场地：15000元
- 餐饮：20000元
- 奖品：10000元
- 其他：5000元
```

## 实际例子

### 找灵感

**你说：**
```
帮我策划今年公司年会，预算6万，100人左右。
先看看去年的年会记录，然后搜搜有什么创意方案。
```

**Mory 会：**
```
好，先看看去年怎么办的...

[读取] 工作/活动策划/2023年会总结.md
[网络搜索] 2024年企业年会创意方案

根据去年反馈和最新趋势，有这些建议：

## 可以改进的地方
1. 增加互动环节（去年反馈时间赶）
2. 加点团队游戏（提升参与感）
3. 搞个主题（更有记忆点）

## 创意方向
1. **主题年会**：复古 disco / 赛博朋克 / 中国风
2. **互动环节**：
   - 线上实时投票
   - 团队闯关游戏
   - 年度记忆墙
3. **惊喜环节**：
   - 盲盒抽奖
   - 员工走红毯
   - 年度大片播放

想往哪个方向展开？
```

### 预算超了？

**你说：**
```
不错！但预算可能超了，哪里能省点？
```

**Mory 会：**
```
好，我来看看哪里能压缩：

## 省钱建议

### 1. 场地费用（可省 3000-5000）
- 选工作日：比周末便宜 20%
- 下午场：比晚场便宜
- 自带部分装饰：减少场地布置费

### 2. 装饰费用（可省 2000-3000）
- 灯笼、拱门：淘宝买 vs 租
- 发动员工做手工装饰
- 配饰从每人一套改成拍照区共用

### 3. 餐饮费用（可省 5000）
- 自助餐形式：200元/人
- 只提供基础酒水

### 优化后预算
| 项目 | 原预算 | 优化后 |
|------|--------|--------|
| 场地 | 18000 | 15000 |
| 餐饮 | 25000 | 20000 |
| 装饰 | 8000 | 5000 |
| 奖品 | 6000 | 6000 |
| 其他 | 3000 | 3000 |
| **合计** | **60000** | **49000** |

这样能省出约 1 万备用金，怎么样？
```


```
根据方案生成详细采购清单，最好带参考链接
```

### 活动复盘模板

```
活动结束后，帮我生成一份复盘模板
```

## 经验总结

1. **记录历史活动**：给以后策划积累素材
2. **保存反馈意见**：知道什么好使、什么要改
3. **多轮对话**：方案可以一点点完善
4. **跟踪执行**：用待办清单追踪进度
