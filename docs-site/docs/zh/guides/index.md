## 先说几个通用技巧

### 1. 先积累，再利用

Mory 的能力建立在你的笔记上。记得越多，它能帮你做的事就越多。

**建议：**
- 养成随手记的习惯
- 不用写得多完美，几个关键词也行
- 后面 Mory 可以帮你整理

### 2. 把需求说清楚

告诉 Mory：
- 你想干嘛
- 希望什么格式
- 有什么特殊要求

**示例：**
```
帮我准备下周的英语考试。
要求：
1. 看看我之前的学习笔记
2. 重点关注语法
3. 给我个每日复习计划
4. 最好用表格
```

### 3. 复杂的事分步来

```
先帮我看看笔记里都有什么主题
```

```
好，针对"时间管理"这块，帮我整理一下要点
```

```
最后生成一份总结，存成新笔记
```

### 4. 不满意就继续改

Mory 第一次的输出可能不够完美，没关系，继续聊：

```
这份计划不错，但第二周能安排得更紧凑点吗？
```

```
表格里的时间改成具体日期吧
```
