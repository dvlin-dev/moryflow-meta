# 构建个人知识库

用 MoryFlow 打造你的「第二大脑」，让知识积累产生复利。

## 为什么用 MoryFlow 做知识库

### 传统知识库的问题

- **收藏 = 遗忘**：收藏了一堆文章，再也没看过
- **散落各处**：知识东一块西一块，没法关联
- **找不到**：想找的时候找不到
- **没消化**：原样保存，没有变成自己的

### MoryFlow 能解决

- **Mory 帮你整理**：自动总结、分类、关联
- **自然语言搜索**：不用记关键词，问就行
- **知识激活**：Mory 能基于你的知识库回答问题
- **越用越聪明**：笔记越多，Mory 越懂你

## 知识库怎么组织

### 推荐结构

```
知识库/
├── 输入/                    # 原始素材
│   ├── 文章摘录/
│   ├── 读书笔记/
│   └── 会议记录/
├── 加工/                    # 整理后的知识
│   ├── 主题笔记/
│   ├── 概念卡片/
│   └── 方法论/
├── 输出/                    # 创作内容
│   ├── 博客文章/
│   ├── 分享材料/
│   └── 项目方案/
└── 索引/
    └── 知识地图.md
```

### PARA 方法（可选）

如果你喜欢 PARA 方法：

```
知识库/
├── Projects/      # 当前项目
├── Areas/         # 长期关注领域
├── Resources/     # 感兴趣的主题
└── Archives/      # 已完成/不活跃的
```

## 知识输入

### 1. 文章摘录

看到好文章，记下核心要点：

```markdown
# 《如何高效学习》摘录

**来源**：https://example.com/article
**日期**：2024-01-15
**标签**：#学习方法 #效率

## 核心观点
1. 主动学习比被动学习效率高 5 倍
2. 间隔重复是记忆的关键
3. 教授他人是最好的学习方式

## 我的想法
这和费曼学习法的理念很像...

## 打算试试
- [ ] 用费曼学习法学习 XXX
```

### 2. 读书笔记

```markdown
# 《书名》读书笔记

**作者**：XXX
**读完时间**：2024-01
**评分**：⭐⭐⭐⭐⭐

## 一句话说明白
这本书讲的是...

## 每章要点
### 第1章：XXX
- 要点1
- 要点2

## 触动我的话
> "引用原文..."

## 读后要做的
- [ ] 试试书里的方法
```

## 知识加工

### 让 Mory 帮你整理

**你说：**
```
帮我整理一下"知识库/输入"文件夹里关于"学习方法"的内容，
生成一份主题笔记。
```

**Mory 会：**
```
[搜索] 知识库/输入/**/*.md 中包含"学习"的文件
[读取] （读取匹配的文件）

根据你的素材，整理了一份学习方法主题笔记：

---

# 学习方法论

## 核心原则

### 1. 主动学习 > 被动学习
- **来源**：《如何高效学习》
- **要点**：主动思考、提问、实践，效率提升 5 倍
- **怎么做**：学完试着给别人讲

### 2. 间隔重复
- **来源**：《学习之道》、Anki 方法
- **要点**：遗忘曲线，定期复习
- **怎么做**：用 Anki 或定期回顾笔记

### 3. 费曼学习法
- **来源**：费曼、《如何高效学习》
- **步骤**：
  1. 选个概念
  2. 用简单话解释
  3. 找出不懂的地方
  4. 简化和打比方

## 我的行动清单
- [ ] 每周整理学习笔记
- [ ] 用费曼法学新东西
- [ ] 建个人 Anki 卡片库

---

存到"知识库/加工/主题笔记/"？
```

### 概念卡片

把重要概念做成卡片：

```markdown
# 费曼学习法

**是什么**
通过"教别人"的方式学习，确保真正理解。

**怎么做**
1. 选个要学的概念
2. 假设要教给一个小孩
3. 找出解释不清的地方
4. 回去学，直到能简单解释

**关联概念**
- 主动学习
- 知识内化
- 教学相长

**什么时候用**
- 学新技术
- 准备演讲
- 检验自己是否真懂了
```

## 知识检索

### 自然语言搜索

不用记关键词，直接问：

```
之前看过一篇关于学习方法的文章，提到什么 5 倍效率，帮我找找
```

```
关于项目管理，我有哪些笔记？
```

### 知识问答

基于你的知识库回答问题：

```
根据我的学习笔记，总结一下有效学习的三个核心原则
```

```
我想学个新技能，根据我之前的笔记，你有什么建议？
```

## 知识输出

### 写博客

```
根据我关于"学习方法"的笔记，帮我写篇博客，
标题叫《我践行的 3 个高效学习方法》
```

### 做分享

```
要做一个关于时间管理的分享，帮我根据相关笔记生成 PPT 大纲
```

### 知识地图

```
帮我生成一份知识库的索引，列出所有主题和关键笔记
```

## 知识库维护

### 定期整理

建议每周花 30 分钟整理：

```
帮我看看这周新增了哪些笔记，有哪些需要整理或关联的
```

### 找出孤岛

```
帮我找出知识库里"孤立"的笔记，就是没被其他笔记引用过的
```

### 清理过时内容

```
帮我找出超过 1 年没更新的笔记，我看看要不要归档
```

## 经验总结

### 1. 降低输入门槛
- 不求完美，先记下来再说
- 用模板减少思考负担
- 碎片时间快速记录

### 2. 定期加工
- 原始素材 → 主题笔记 → 概念卡片
- 每周整理，别堆着
- Mory 帮忙，但要有自己的思考

### 3. 积极输出
- 输出是最好的学习
- 分享倒逼你整理
- 知识用起来才有价值

### 4. 持续迭代
- 知识库是活的，要持续更新
- 定期回顾修正
- 过时的就删掉
