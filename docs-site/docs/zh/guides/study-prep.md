# 用 MoryFlow 准备考试

让 Mory 当你的学习助手，帮你定计划、整理知识、查漏补缺。

## 适用场景

要准备一场重要考试（四六级、考研、考证什么的），想要：

- 定个合理的复习计划
- 整理和巩固知识点
- 薄弱环节重点突破
- 了解最新考试资讯

## 准备工作

### 1. 建个学习文件夹

```
学习/
├── 英语/
│   ├── 单词本.md
│   ├── 语法笔记.md
│   └── 听力练习.md
└── 复习计划/
```

### 2. 平时记录学习笔记

学习的时候把笔记记在 MoryFlow 里：

**单词本.md 示例：**
```markdown
# 英语四级单词

## Unit 1
- abandon v. 放弃，遗弃
- ability n. 能力
- abroad adv. 在国外

## Unit 2
...
```

**语法笔记.md 示例：**
```markdown
# 语法要点

## 时态
### 现在完成时
- 结构：have/has + 过去分词
- 用法：表示过去发生的动作对现在有影响
- 例句：I have finished my homework.
...
```

## 实际例子

### 让 Mory 帮你定计划

**你说：**
```
下个月15号考英语四级，帮我定个复习计划。
先看看我的英语学习笔记，了解下我学到哪了。
```

**Mory 会：**
```
好的，我来看看你的笔记...

[读取] 学习/英语/单词本.md
[读取] 学习/英语/语法笔记.md
[读取] 学习/英语/听力练习.md

看了一下，你的情况是：
- 单词积累了大概 500 个
- 时态语法掌握得不错，但从句部分笔记比较少
- 听力练习记录不多

给你定个 30 天计划：

| 周次 | 重点 | 每天要做的 |
|------|------|----------|
| 第1周 | 单词 | 复习50个旧词，新学30个词 |
| 第2周 | 语法 | 主攻从句，每天做语法题 |
| 第3周 | 听力 | 每天30分钟听力训练 |
| 第4周 | 综合 | 做真题，查漏补缺 |

要我把这个计划存起来吗？
```

### 搜搜最新考点

**你说：**
```
帮我搜一下今年四级考试有什么新变化
```

**Mory 会：**
```
[网络搜索] 2024年英语四级考试变化

搜到了，今年有这些更新：

1. **听力部分**：新闻听力题变多了，建议多练 BBC/VOA
2. **阅读部分**：信息匹配题难度稍微提升了
3. **写作部分**：更注重图表描述

建议你：
- 每天听15分钟英语新闻
- 多练信息匹配题
- 准备几个图表作文模板

要我把这些记到你的笔记里吗？
```

### 整理知识点

**你说：**
```
把语法笔记里的从句部分整理成思维导图格式
```

**Mory 会：**
```
[读取] 学习/英语/语法笔记.md

整理好了：

# 英语从句思维导图

## 名词性从句
├── 主语从句
│   └── That he came late surprised us.
├── 宾语从句
│   └── I know that he is honest.
├── 表语从句
│   └── The fact is that we have lost.
└── 同位语从句
    └── The news that he won is exciting.

## 定语从句
├── 限制性定语从句
│   └── The book which you gave me is good.
└── 非限制性定语从句
    └── My brother, who is a doctor, lives in Beijing.

## 状语从句
├── 时间状语从句 (when, while, as)
├── 条件状语从句 (if, unless)
├── 原因状语从句 (because, since)
└── 让步状语从句 (although, though)

存成新笔记？
```

### 来个小测验

**你说：**
```
根据我的单词本，出10道选择题考考我
```

**Mory 会：**
```
[读取] 学习/英语/单词本.md

好，根据你背过的单词出题：

**1. The company decided to ___ the project due to budget issues.**
A. abandon  B. ability  C. abroad  D. absolute

**2. She has the ___ to solve complex problems.**
A. abandon  B. ability  C. abroad  D. absence

...

答完告诉我，我帮你批改，错的地方给你讲讲。
```

## 更多技巧

### 定期复习

```
每周日帮我回顾这周背的单词，列个复习清单
```

### 错题整理

```
刚做错一道定语从句的题，帮我记到错题本，分析下为什么错
```

### 追踪进度

```
统计下我这个月学习情况，看看计划完成了多少
```

## 经验总结

1. **坚持记录**：每天学了啥都记下来，Mory 才能更好地帮你
2. **及时整理**：学完一章就让 Mory 帮你做个总结
3. **定期测试**：让 Mory 出题考你，检验效果
4. **查漏补缺**：让 Mory 分析你的薄弱点
