# 用 MoryFlow 规划旅行

让 Mory 当你的私人旅行顾问，打造专属行程。

## 适用场景

计划一次旅行，想要：

- 了解目的地最新信息
- 定个合理的行程
- 符合自己的喜好和预算
- 记录旅途中的点滴

## 准备工作

### 1. 建个旅行文件夹

```
生活/
├── 旅行/
│   ├── 我的旅行偏好.md
│   ├── 2023日本游.md
│   └── 旅行愿望清单.md
└── ...
```

### 2. 写下你的旅行偏好

**我的旅行偏好.md 示例：**
```markdown
# 我的旅行偏好

## 住宿
- 预算：300-500元/晚
- 偏好：干净、交通便利、有早餐
- 不喜欢：青旅、太偏僻的地方

## 饮食
- 喜欢：当地特色、街边小吃
- 不吃：香菜、海鲜过敏
- 预算：每天200左右

## 行程风格
- 不想太赶，每天2-3个景点就够了
- 喜欢慢悠悠地逛当地市场
- 对博物馆、历史建筑感兴趣
- 不太喜欢网红打卡点

## 其他
- 一般自由行
- 喜欢拍照，会考虑光线好的时间
```

### 3. 记录以前的旅行

把之前的旅行记下来，Mory 就能更懂你的喜好了。

### 找住宿

**你说：**
```
帮我搜搜京都住宿推荐，符合我的预算和偏好
```

**Mory 会：**
```
[网络搜索] 京都住宿推荐 300-500元 交通便利

按你的要求（300-500元/晚、交通便利、有早餐），推荐这几个区域：

## 推荐住宿区域

### 1. 四条河原町（最推荐）
- 优点：市中心，交通枢纽，吃喝购物方便
- 缺点：稍微吵
- 推荐：
  - Mitsui Garden Hotel 四条 (~450元)
  - Hotel Gracery 京都 (~400元)

### 2. 京都站周边
- 优点：交通最方便，去岚山/奈良都近
- 缺点：商业化，京都味淡一些
- 推荐：
  - Daiwa Roynet Hotel (~380元)
  - Hotel Keihan 京都 (~350元)

### 3. 祇园/东山
- 优点：京都味最浓，离景点近
- 缺点：价格略高
- 推荐：
  - Hotel The Celestine 京都祇园 (~500元)

## 我的建议
看你的行程（东山、岚山、伏见），建议住**四条河原町**，去哪都方便，晚上还能逛先斗町、锦市场。

要存下来吗？
```

## 经验总结

1. **记下你的偏好**：Mory 推荐的行程就更对味
2. **随时记录**：路上随手记，别等回来再补
3. **保存有用的攻略**：存成笔记，下次能用
4. **旅行复盘**：总结经验，每次旅行都更好
