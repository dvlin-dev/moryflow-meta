# 设置

MoryFlow 的设置面板让你可以按自己的喜好来调整。

## 打开设置

点击右上角的 **齿轮图标** 就行。

<!-- 截图占位符：设置入口 -->
![设置入口](/images/settings/settings-entry.png)

## 有哪些设置

### 常规

基础的配置：

- **笔记文件夹路径**：你的笔记存在哪
- **主题**：深色还是浅色
- **语言**：界面语言

### AI 模型

配置你想用的 AI 模型：

- 填各家服务商的 API Key
- 选默认用哪个模型
- 添加自定义模型

[了解 AI 模型配置 →](/zh/settings/ai-models)

### Ollama 本地模型

想完全离线用 AI？可以用 Ollama：

- 配置 Ollama 连接
- 下载和管理本地模型
- 完全离线，数据不出你电脑

[了解 Ollama 配置 →](/zh/settings/ollama)

### 自定义服务商

添加其他兼容 OpenAI 协议的服务：

- 配置自定义 API 端点
- 设置认证信息
- 选择可用模型

[了解自定义服务商 →](/zh/settings/custom-provider)

### MCP 扩展

给 Mory 加技能：

- 添加和配置 MCP 服务器
- 查看可用工具
- 管理连接状态

[了解 MCP 扩展 →](/zh/features/mcp)

### 主题

个性化界面：

- 深色/浅色模式
- 跟随系统设置

[了解主题设置 →](/zh/settings/theme)

## 设置存在哪

MoryFlow 的设置存在本地，不会上传到云端。

设置文件位置：
- **macOS**：`~/Library/Application Support/MoryFlow/`
- **Windows**：`%APPDATA%/MoryFlow/`

:::tip 小贴士
如果设置出问题了，可以删掉设置文件然后重启应用，就会恢复默认配置。
:::
