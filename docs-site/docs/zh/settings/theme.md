# 主题设置

MoryFlow 支持深色和浅色两种主题，随你喜欢。

## 怎么切换

### 方法一：在设置里

1. 打开设置
2. 在「常规」里找到「主题」
3. 选一个：
   - **浅色**：白天或亮堂的地方用
   - **深色**：晚上或暗的地方用
   - **跟随系统**：自动跟着系统走

<!-- 截图占位符：主题设置 -->
![主题设置](/images/settings/theme-settings.png)

### 方法二：快捷键

- **macOS**：`⌘ + Shift + L`
- **Windows**：`Ctrl + Shift + L`

## 长什么样

## 跟随系统

选了「跟随系统」之后，MoryFlow 会自动跟着系统的主题走：

- **macOS**：跟系统偏好设置里的「外观」
- **Windows**：跟系统设置里的「颜色」模式

### 自动切换

如果你的系统设置了日落后自动切换深色模式，MoryFlow 也会跟着切。

## 编辑器也会变

编辑器里的代码高亮也会跟着主题变：

- 浅色主题：亮色代码配色
- 深色主题：柔和代码配色

## 常见问题

### 切换后有些地方没变

重启一下应用就好了。

### 想要更多主题

现在只支持浅色和深色。以后可能会加更多。

### 护眼模式

MoryFlow 的深色主题已经做了护眼优化：
- 不用纯黑背景
- 文字对比度适中
- 减少蓝光

想要更强的护眼效果，可以配合系统的 Night Shift（macOS）或夜间模式（Windows）一起用。
